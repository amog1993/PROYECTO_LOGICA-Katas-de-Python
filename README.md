# 🐍 Proyecto Katas de Python

## Descripción

Este proyecto reúne una serie de katas de programación en Python enfocadas en desarrollar habilidades fundamentales como:

- Manipulación de datos
- Programación funcional (map, filter)
- Manejo de excepciones
- Recursividad
- Lógica algorítmica

## 🎯 Objetivos

- Reforzar la lógica de programación
- Practicar estructuras de datos (listas, diccionarios, tuplas)
- Aplicar buenas prácticas en Python
- Comprender el manejo de errores y excepciones
- Introducir conceptos de programación funcional

## Contenido de las Katas

El proyecto incluye ejercicios como:

- Frecuencia de letras en un texto  
- Transformación de listas con map()  
- Filtrado de datos con filter()  
- Manejo de divisiones con control de errores  
- Cálculo de medias y validación de resultados  
- Cálculo de factorial (recursividad)  
- Filtrado de mascotas prohibidas  
- Manejo de excepciones personalizadas  
- Validación de entrada del usuario  

## Tecnologías utilizadas

- Python 3
- Jupyter Notebook

## Ejemplo de uso

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

print(factorial(5))  # Output: 120
